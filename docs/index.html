XJTLU Timetable Bookmarklet

请在 iPhone Safari 打开本页。复制下面的 bookmarklet URL，然后把它粘贴到一个 Safari 书签的网址栏里。

复制 bookmarklet URL

安装步骤

  1. 在 iPhone Safari 打开本页。
  2. 复制上方 bookmarklet URL。
  3. 在 Safari 中添加任意网页为书签。
  4. 进入书签列表，点“编辑”。
  5. 把书签名称改成 Export XJTLU Timetable。
  6. 删除原网址，粘贴 bookmarklet URL。
  7. 保存书签。

使用步骤

  1. 打开 XJTLU e-Bridge 课表页面。
  2. 确认课表已经显示。
  3. 点 Safari 书签，选择 Export XJTLU Timetable。
  4. 输入 Week 1 Monday 和 Last Week。
  5. 保存或分享生成的 .ics 文件。
